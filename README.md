bisheng-theme-ui
================
> react ui library theme for [bisheng](https://github.com/benjycui/bisheng). Almost Copied from [react-amap](https://github.com/ElemeFE/react-amap)

Usage
-----

```bash
$ yarn add bisheng-theme-ui --dev
```

```javascript
// bisheng.config.js
{
  theme: 'bisheng-theme-ui'
  ....
}
```
